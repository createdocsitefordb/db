# :material-database: Log Ingestion into Databahn

Databahn ingests logs from three primary connectors/sources:

* [Network logs via Palo Alto Networks (Syslog)](/paloalto/)
* [Identity logs via Microsoft Entra ID (API)](/datasources/entraid/)
* [Endpoint logs via CrowdStrike (AWS SQS/S3)](/db/datasources/crowdstrike/)

Each category has specific access, configuration, and network requirements.
