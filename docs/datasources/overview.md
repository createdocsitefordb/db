# :material-database: Log Ingestion into Databahn

Databahn ingests logs from three primary connectors/sources:

* [Network logs via Palo Alto Networks (Syslog)](/paloalto.md)
* [Identity logs via Microsoft Entra ID (API)](/datasources/entraid.md)
* [Endpoint logs via CrowdStrike (AWS SQS/S3)](/db/datasources/crowdstrike.md)

Each category has specific access, configuration, and network requirements.
