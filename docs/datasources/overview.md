# :material-database: Log Ingestion into Databahn

Databahn ingests logs from three primary connectors/sources:

* [Network logs via Palo Alto Networks (Syslog)](https://createdocsitefordb.github.io/db/datasources/paloalto/)
* [Identity logs via Microsoft Entra ID (API)](https://createdocsitefordb.github.io/db/datasources/entraid/)
* [Endpoint logs via CrowdStrike (AWS SQS/S3)](https://createdocsitefordb.github.io/db/datasources/crowdstrike/)

Each category has specific access, configuration, and network requirements.
